# Installation

Clonez du git :

    git clone https://github.com/corentinft/ProjetMVC.git

Allez dans le dossier du projet :

    cd ProjetMVC

installez le projet laravel:

    composer install

    npm install

    npm run dev

lancez le projet :

    php artisan serve
